Welcome to Apache2 Server

Basic Introduction

Apache2 is a open source web server software. With the help of apache you can create and run webserver on your device.

Instruction

  • Run apachectl to start Apache2 Server
  • In this installation I change Document Path /sdcard/htdocs. So you can access all your web Documents easily. And use external coding apps for creating Web Pages.
  • Web Server url is still http://localhost:8080/
